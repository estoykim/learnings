Link Elements

There are four ways to create a link on your webpage

  • Links from one website to another
  • Links from one page to another on the same website
  • Links from one part of a web page to another part of the same page
  • Links that open in a new browser window
  • Links that start up your email program and address a new email to someone

Linking to Other Sites

To link to other sites, you use the a href tag and use the URL for the link.

Linking to Other Pages on the Same Site

To link other pages on the same site, use relative URL. Use the name of the text file. If it is inside a folder, for example, the file is inside the image folder, then use /image/index.html to go to your Homepage. It is also called External Links

TIP: One of the most common reasons a web page author might want a link to be opened in a new window is if it points to another website. In such cases, they hope the user will return to the window containing their site after finishing looking at the other one.

Linking to a Specific Part of the Same Page

To link to a certain part of your web page. Use id attribute to the header tag and insert appropriate value. To add the value, for example you are at the top of your web page and want to jump to the bottom, put an a href HTML tag first at the TOP and put the value with #, say ="#bottom". Then add a value to the header tag you want to link it to. To do that, put id value of the link and it should be ="bottom". Always remember that the link should have a id attribute value with #.

Always remember that a link shoud have an a tag with href attribute, and put id attribute to the header you want to link it to. The value of the href attribute should always have # followed by your preferred link name
