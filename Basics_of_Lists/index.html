List Element

Ordered, Unordered, Definition, and Nested List

There are four different ways to show list on your website.

  1. Ordered List
  2. Unordered List
  3. Definition List
  4. Nested List

Ordered List (ol, li)

For example:

  1. Heat the pan.
  2. Put cooking oil.
  3. Chop the garlic and onions.
  4. Ready to eat.
^

Unordered List (ul, li)

- with a shape of circle as a default shape for bullet point.

For example:

  • Potatoes
  • Carrots
  • Rice
  • Tomatoes

- with type attribute specifying different shapes such as circle, square, diamonds, and so on, or you can style the type of bullet point using CSS.

  • Coffee
  • Sugar
  • Milk
^

Definition List (dl, dt, dd)

For example:

Social Media Manager
- Someone who manages social media accounts for branding and engagements.
Web Developer
- Responsible for making a website using HTML, CSS, and Javascript.
- Someone who can also maintain and fix bugs everytime an update is made.
^

Nested List

- A list within a list

For example:

  • Electric Motor
    • Stator
    • Rotor
      • Squirrel Cage Rotor
      • Wound Rotor
    • Armature
  • Electric Generator
    • Prime Movers
      • Geothermal
      • Diesel
      • Nuclear Energy
