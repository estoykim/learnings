How Directories Work

Root Folder

  • A root folder contains all the files of your website.
  • In the example picture, it contains a file called index.html which is the homepage for the entire site.
  • It also contains individual folders such as, movies, music, and theatre sections of the site.

Absolute URL

  • It is the full web address of the site.
    • Eg. https://www.examplesite.com
      • the http is not encrypted and it is not safe.
      • the https is more secure.
